🛠️ 캐리마텍 데이터 관리자

이 페이지는 관리자 편집 전용입니다. 고객에게 공유되는 화면(index.html)이 아닙니다.

📌 사용 순서

1. 아래에서 수치·설명을 수정하거나 소재를 추가/삭제합니다. (문법 신경 쓸 필요 없음)

2. 맨 아래 “data.js 내려받기” 버튼을 눌러 새 data.js 파일을 받습니다.

3. GitHub 저장소의 assets/data.js를 받은 파일로 교체(업로드)합니다.

※ 이 페이지에서 “저장”해도 서버에 자동 반영되지 않습니다. 반드시 3번까지 해야 고객 화면에 적용됩니다.
